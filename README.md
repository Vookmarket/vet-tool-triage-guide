# 電話トリアージガイド (VT-002)

動物病院の受付スタッフ・看護師が、飼い主からの電話中に使う **獣医療緊急度判定ツール** です。

ルールベース（デシジョンツリー）で即時判定。LLM不使用、外部依存なし、オフラインで動作します。

## 動作要件

- Python 3.10+
- 外部パッケージ不要（標準ライブラリのみ）
- OS非依存（macOS / Windows / Linux）

## 使い方

### 対話モード

```bash
python3 tool.py
```

対話形式で以下を入力します：

1. 動物種（犬/猫）
2. 年齢区分（子犬子猫/成犬成猫/高齢）
3. 発症からの経過時間
4. 症状（カテゴリ選択 or テキスト入力）
5. 追加メモ

### JSON出力モード

```bash
python3 tool.py --json
```

判定結果をJSON形式で出力します。外部システムとの連携に便利です。

### テスト実行

```bash
python3 -m pytest test_tool.py -v
```

## モジュール構成

| `test_tool.py` | テストスイート |

## 緊急度判定基準

### 🔴 RED — すぐ来院

- 呼吸困難、開口呼吸、チアノーゼ
- 意識消失、けいれん重積
- 大量出血、交通事故、高所落下
- 猫の排尿不能（尿路閉塞）
- 犬の胃捻転疑い（腹部膨張+嘔吐）
- 熱中症、低体温
- 致死性毒物摂取
- 子犬・子猫の元気消失、ミルク拒否

### 🟡 YELLOW — 本日中に来院

- 24時間以上の嘔吐・下痢
- 食欲不振24時間以上
- 咬傷、骨折疑い
- 毒物摂取
- 運動失調、震え
- 血尿、排尿困難

### 🟢 GREEN — 様子見OK

- 1回の嘔吐（元気あり）
- 軽い咳
- 軽度の目やに
- 軽度の掻痒

### 安全側設計

- 複数の軽度症状が組み合わさると自動で緊急度が上がります
- 子犬・子猫は軽度症状でもYELLOW以上に判定されます
- 24時間以上経過した症状は緊急度が上がります
- 症状が特定できない場合はYELLOW（安全側）に判定されます

## 情報源

本ツールの症状分類・緊急度判定基準は、以下の獣医学文献に基づいています：

- **Merck Veterinary Manual** — XABCDE primary survey
- **PMC NIH** — 色分けトリアージシステム (Red/Orange/Yellow/Green)
- **Veterinary Emergency of Midlothian** — 5段階コード (White/Green/Yellow/Red/Black)
- **森田動物医療センター** — 日本語症状分類（呼吸器、神経・意識、消化器、外傷、眼科、温度異常、犬猫特有症状）

## 免責事項

このツールは電話トリアージの参考補助を目的としており、獣医師による診断の代替ではありません。最終的な判断は必ず獣医師が行ってください。緊急性に疑いがある場合は、常に安全側（より高い緊急度）に判断してください。
